• О программе АвтоСалон

В настоящее время на российском рынке практически отсутствует программное обеспечение, предназначенное для предприятий, торгующих автомобилями или номерными агрегатами. Все используемое фирмами программное обеспечение на немецком или английском языках и не адаптировано для российского рынка. Система "Автосалон" решает эту проблему.

Система предназначена для автоматизации складского учета автомобилей отечественного и импортного производства.


Возможности системы:
  • Третья версия системы является продолжением серии одноименных программных продуктов, созданных специалистами нашей компании и написана с учетом пожеланий и рекомендаций пользовтелей.
  • Сетевая платформа "Клиент-Сервер" (FireBird). Автоматическое отслеживание обновления информации на уровне Базы Данных (без использования таймера).
  • Изменения в Интерфейсе в значительной степени облегчают общение пользователя с программой. Например, при перемещении по Складу или Журналу можно видеть подробную информацию о текущем автомобиле (идентификационные данные, сведения о документах, информация о покупателе и комитенте и т.д.).
  • Сколько угодно много Предприятий (торгующих организаций).
    • Краткиое и полное наименование;
    • Банковские реквизиты;
    • Сведения о лицензии;
  • Полная Мультивалютность. Возможность установки входных и отпускных цен в любой валюте.
    • Статистика курса валюты;
    • Диаграммы статистики курсов валюты (до пяти графиков одновременно);
  • Ведение различных Справочников:
    • Справочник по автомобилям;
    • Гибкая настройка списка комплектации автомобилей. Список является динамическим, т.е. Вы можете его сами настроить как требуется;
    • Справочник по предприятиям;
    • Справочник по валютам;
    • Справочник по сотрудникам;
    • Справочник по покупателям;
    • Справочник по комитентам;
    • Справочник по номерам счет-справок;
    • Справочник по транзитным номерам;
    • Справочник по источникам информации;
    • Справочник по цветам автомобиля;
    • Справочник по странам-производителям;
    • Справочник по видам оплаты;
    • Справочник по фирмам-производителям;
    • Справочник по таможням;
  • Ведение Склада предприятия.
    • Различные виды сортировки;
    • Широкие возможности поиска;
    • Печать развернутой информации об автомобиле;
    • Печать прайс-листа;
    • Просмотр подробной информации по автомобилю;
    • Оформление акта оценки.
  • Ведение Журнала продаж.
    • Различные виды сортировки;
    • Различные виды отображения (фильтрации) информации;
    • Широкие возможности поиска;
    • Печать развернутой информации об автомобиле;
    • Печать прайс-листа;
    • Оформление документов в любое время;
    • Возможность возврата автомобиля на склад;
    • Просмотр подробной информации по автомобилю;
    • Оформление договора на оказание услуг.
  • Печать Справки-Счет на стандартном бланке (два вида: для автомобилей и для тракторов).
  • Печать ПТС (паспорт транспортного средства) на стандартном бланке.
  • Печать Транзитного номера на стандартном бланке.
  • Печать бухгалтерских документов: Накладная, Счет-фактура, Счёт на оплату.
  • Печать Актов, Талонов, Договоров.
  • Отчеты и диаграммы:
    • О приходе за любой период;
    • О реализации за любой период;
    • О движении за любой период;
    • По остаткам на любой день;
    • По исполнителям (например, продавцам-консультантам);
    • Анализ рекламных кампаний;
    • Отчеты о формах оплаты;
    • Анализ спроса по автомобилям (по всем / по реализованным);
    • Реестр использованных счет-справок;
    • Реестр использованных транзитных номеров;
    • Отчет <Объединенный> - отчет по использованным справкам-счет и транзитным номерам (все в одном отчете).
  • Статистика с возможностью формирования диаграмм:
    • Статистика по автомобилям;
    • Статистика по предприятиям;
    • Статистика по покупателям;
    • Статистика по комитентам;
    • Статистика по источникам информации;
    • Статистика по цветам;
    • Статистика по странам производителям;
    • Статистика по видам оплаты;
    • Статистика по фирмам-изготовителям;
    • Статистика по таможням.
  • Описание работ и нормы-времени на Предпродажную подготовку по отечественным автомобилям.
  • Встроенный Текстовый редактор по возможностям не уступающий Microsoft WordPad. Вы можете формировать любые сопутствующие документы (договоры, акты и т.п.).
  • Встроенный Проводник Windows.
  • Подробная Справочная система и Советы дня.

В комплект поставки каждого дистрибутива входят:
  • Руководство пользователя;
  • Регистрационная карта (лист);
  • Компакт-диск с программой.

• Требования и рекомендации

  • Операционная система Windows 98,ME,NT,2000.
  • Процессор Intel Pentium
  • Оперативна память 16 Мб (минимум)
  • SVGA монитор (True color, 1024x768 желательно)
  • Свободный объем на жестком диске 30 Мб
  • Привод CD-ROM (для установки системы на компьютер)
  • Мышь

• Прайс-лист

Стоимость системы "АвтоСалон" 2 950 рублей.

Цена указана с учётом почтовой доставки по России, в другие страны доставка оплачивается дополнительно.
