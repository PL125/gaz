О программе АвтоНормы.

Система содержит трудоемкости работ по техническому обслуживанию и ремонту автомобилей иностранного и отечественного производства.
Система "АвтоНормы" постоянно развивается и периодически пополняется новыми нормативая, появляется новая функциональность.

Нормативы:
  • Нормы времени по модельному ряду автомобилей ВАЗ.
  • Нормы времени на слесарный ремонт иностранных автомобилей.
  • Нормы времени на кузовной ремонт иностранных автомобилей.
  • Нормы времени по модельному ряду автомобилей ИЖ.
  • Нормы времени по модельному ряду автомобилей УАЗ.
  • Нормы времени по автомобилям ГАЗ (3102, 3110, Соболь).
  • Нормы времени по грузовым автомобилям ГАЗ, ЗИЛ, КамАЗ, МАЗ.
  • Нормы времени по автобусам ПАЗ, ЛиАЗ, ГолАЗ, ГАЗель (микроавтобус), Mercedes Benz.
Возможности:
  • Возможность калькуляции стоимости работ.
  • Возможность установки стоимости нормо-часа для каждой работы.
  • Возможность установки кратности (коэффициента) для каждой работы.
  • Возможность добавления в калькуляцию сопутствущих работ.
  • Справочник по общим работам.
  • "Менеджер пользователей". Возможность запретить пользователю изменение нормативов.
  • Возможность вставлять в отчетный документ ФИО клиента и наименование его автомобиля.
  • Возможность экспорта калькуляции в различные форматы (BMP, JPEG, TIFF, TXT, HTML, XLS, XML, RTF, PDF, GIF, TXT, CSV, E-Mail).

Требования и рекомендации.

Программные средства:
Система "АвтоНормы 3" тестировалась на следующих операционных системах:

  • Windows® 98 Second Edition (не рекомендуется)
  • Windows® ME (не рекомендуется)
  • Windows® NT 4.0 +Service Packs 6 (не рекомендуется)
  • Windows® 2000 Professional +Service Packs 4 (рекомендуется)
  • Windows® XP Professional or Home Edition +Service Packs 2 (рекомендуется)
  • Windows® Server 2003 +Service Packs 1 Enterprise Edition

Для работы системы "АвтоНормы 3" требуется установка следующего программного обеспечения:
  • Firebird Server Win32 build, v.1.5.1.4481 (входит в поставку)
  • Microsoft Internet Explorer 4 и выше (необходим для справочной системы)

Аппаратные средства:
  • Процессор семейств Intel® (Pentium® / Celeron® / XeonT), AMD (AthlonT / DuronT) тактовая частота которых составляет не менее 200 МГц.
    Работа на процессорах ADM K5 и AMD K6 не поддерживается.

  • Минимальные требования к оперативной памяти зависят от используемой операционной системы: для Windows XP/2000/NT 4.0 - 64 Мб; Windows Me/98 - 32 Мб. При работе в многопроцессорных системах требуется дополнительно 16 Мб оперативной памяти для каждого дополнительного процессора.

  • Видеоплата и монитор с разрешением не менее 800х600 точек, глубина цвета не менее 16bit.
    Рекомендуемые параметры: разрешение 1024х768 и более, глубина 16bit и более, мелкий шрифт, стандартная или классическая цветовая схема.

  • Свободное место на жестком диске не менее 100 Mb.

  • Клавиатура.

  • Манипулятор мышь.

  • Для сетевого варианта наличие локальной вычислительной сети. Установленный протокол TCP/IP.

  • CD-ROM привод (обязателен для установки и активации системы на компьютере).
    Внимание! Активация может не пройти в автоматическом режиме на некоторых типах приводов CD-ROM/R/RW и DVD-ROM/R/RW. Свяжитесь с технической поддержкой для консультации по данному вопросу.
