При подключении ромокабеля к компьютеру Windows 7 пытается загрузить драйвер для него с сайта майкрософта.
Это происходит даже при установленных с этого диска драйверах PL2303.
Чтобы этого не происходило, отключите доступ в интернет или нажмите надпись "Пропустить поиск драйвера".
Если Вы установите драйвер с сайта майкрософта, то диагностика работать не будет (чиптюнинг будет).

Для владельцев Windows 8: для Вас подойдет драйвер VCP2.
Некоторые антивирусы опознают его как зараженный. Вирусов там нет!
