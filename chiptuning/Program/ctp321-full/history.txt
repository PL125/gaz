22.12.2003 - официальное обновление: ChipTuningPRO 3.00 build 1095
-------------------------------------------------------------------

!исправлена ошибка, возникающая при анализе перемешанной прошивки.
!исправлена ошибка в инсталляторе, приводящая к невозможности установки программы под Windows98.
!исправлена неправильная работа с перемешанными прошивками на базе серийного ПО.
!исправлено неправильное отображение оси X в таблице "Обороты выхода из режима пуска" для прошивок Bosch MP7.0
!исправлено неправильное отображение на диагностике идентификационных данных в прошивках семейства Январь-5.1/VS-5.1/Bosch M1.5.4, в том случае, если они вводились более короткими, чем предусмотрено.
!исправлены карты калибровок для прошивок серии K17 и L19 (не отображались таблицы, относящиеся к реле блокировки стартера и коэффициент уставки РХХ).
!исправлена ошибка при сохранении файлов прошивок Январь-4.
!исправлена ошибка при открытии файлов в формате hex.
!исправлена ошибка в функции "Инструменты - создать прошивку еепром для Bosch M154" делает прошивку C:\FIRMWAREeeprom.bin (не хватает символа "\" между C:\FIRMWARE и eeprom.bin).

*доработана функция импорта калибровок - теперь импорт работает корректно независимо от того, в какой локализованной версии Windows был создан файл файл экспорта. "Русская" Windows в качестве разделителя в десятичной дроби использует знак ',', а другие версии - знак '.', что приводило к невозможности импорта калибровок из несовместимого файла. Теперь импорт работает коррекно с тем и другим типами разделителей.
*изменено расположение панели редактирования графиков (перенесена под график по просьбам пользователей).
*оптимизирована отрисовка 3D-поверхностей
*улучшена степень компресии файлов в инсталляционной базе, благодяря этому, размер дистрибутива уменьшился на 30%.

+добавлен тип ЭБУ "Микас 7.6", устанавливаемый на Daewoo Lanos, Daewoo Matiz. Опция приобретается дополнительно.
+добавлено отображение координаты Y на графике для курсора мыши
+добавлено отображение дельты между двумя 2D графиками в режиме сравнения (дельта отображается в Status Bar, если навести курсор мыши на одну из точек в режиме сравнения).
+добавлена поддержка новых прошивок для ЭБУ VS-5.1 (V5V05M30, V5V03L25).
+добавлена поддержка новой прошивки для ЭБУ Январь-5.1.3 (J5V26K23)
+добавлена возможность ручного ввода пути к рабочим каталогам в настройках программы (по просьбам пользователей).
+добавлена возможность работы со вторым набором калибровок в сдвоенных прошивках Bosch M1.5.4
+при сравнении двух абсолютно идентичных файлов теперь выводится соотвествующее сообщение, неиспользуемые в обоих файлах калибровки не отображаются при сравнении.
